PRE

ก่อนเริ่มใช้งาน — ตั้งค่า API Key

— 🔑 ตั้งค่า Google API Key ใน Streamlit Secrets จำเป็น · ทำครั้งเดียว

แอปนี้ใช้ Gemini API ของ Google เพื่อสร้าง content และ Imagen 3 เพื่อ Gen ภาพ ต้องมี API Key ก่อนจะใช้งานได้

ขั้นตอนขอ API Key:

1️⃣ ไปที่ https://aistudio.google.com/apikey → Sign in ด้วย Google Account → กด Create API key → Copy key ที่ได้
2️⃣ เปิดไฟล์ .streamlit/secrets.toml ในโปรเจกต์ (ถ้าไม่มีให้สร้างใหม่) แล้วเพิ่มบรรทัดนี้:
GOOGLE_API_KEY = "AIzaSy...xxxxxx" # วาง key ของคุณตรงนี้
3️⃣ ถ้า Deploy บน Streamlit Cloud → ไปที่ Settings → Secrets → วาง GOOGLE_API_KEY = "..." แทน
⚠️ Imagen 3 (Gen ภาพ AI) ต้องการ Google Cloud billing เปิดอยู่ ถ้ายังไม่เปิด แอปจะใช้รูปต้นฉบับที่คุณอัปโหลดแทนให้อัตโนมัติ — pipeline ไม่หยุด
✅ ถ้าตั้งค่าถูกต้อง แอปจะโหลดขึ้นมาปกติ ไม่มี error แดง
— 📦 ติดตั้ง Libraries ที่จำเป็น จำเป็น · ทำครั้งเดียว

รัน command นี้ใน terminal ก่อน run แอป:

pip install streamlit requests gtts

จากนั้น run แอปด้วย:

streamlit run 8_Workflow_Builder_3.py

วางข้อมูลจากเว็บ (ไม่บังคับ แต่แนะนำ)

🌐 WEB DATA ANALYZER
3.1 📋 Copy ข้อมูลจากเว็บ มาวางในกล่อง "ข้อมูลเพิ่มเติมจากเว็บ" ไม่บังคับ

กล่องนี้อยู่ใต้ช่องแบรนด์ใน Product Brief วางข้อมูลดิบอะไรก็ได้ เช่น:

  • Spec จากหน้าสินค้าบนเว็บ
  • รีวิวจากลูกค้าบน Shopee / Lazada / Google
  • ข้อมูลสินค้าคู่แข่งที่อยากเปรียบเทียบ
  • คำถามที่คนถามบ่อยใน comment
  • ราคาจากหลายเจ้า
ℹ️ วิธีเอาข้อมูล: เปิดเว็บสินค้า → Ctrl+A เลือกทั้งหน้า → Ctrl+C → มาวางในกล่องนี้ได้เลย ไม่ต้องจัดรูปแบบ

06

กดปุ่ม RUN CONTENT FACTORY

🏭 RUN
6.1 🏭 กดปุ่มใหญ่ "🏭 RUN CONTENT FACTORY" ขั้นตอนสำคัญ

ปุ่มนี้อยู่ด้านล่างสุดของฝั่งซ้าย สีน้ำเงิน ขนาดใหญ่ กดได้เลย ถ้าข้อมูลครบ

❌ ถ้าปุ่มกดไม่ได้ (สีจาง) แสดงว่ายังขาด: รูปภาพ หรือ ชื่อสินค้า หรือ รายละเอียดสินค้า — ให้กลับไปกรอกให้ครบก่อน

สิ่งที่จะเกิดขึ้นหลังกด:

⚡ Progress bar จะขึ้นมาด้านขวา พร้อม Log Terminal แสดงสถานะ real-time ว่า AI ทำถึง Node ไหนแล้ว
⚡ Pipeline icons บนหน้าจะเปลี่ยนสีตาม: 🟡 Running → 🟢 Done → ⚪ Skip → 🔴 Error
⏱️ ระยะเวลา: ถ้าเปิดทุก Node ใช้เวลาประมาณ 3–8 นาที ขึ้นอยู่กับความเร็ว API ของ Google — อย่าปิดหน้าต่างระหว่างรอ

Pipeline ที่จะรันตามลำดับ:

🖼️ Input› 🧠 Strategy› 🎨 Gen Image› 🎞️ Storyboard› 🎬 Video Script› 🎤 Voice Script› 🔊 TTS Audio› 📣 Ad Copy› 📝 Caption› 📧 Email/LINE› #️⃣ Hashtags› 🧪 A/B Test› 📅 Calendar› ⭐ Review› 🔄 Repurpose› 📦 ZIP

Download ZIP — ขั้นตอนสุดท้าย

🎉 เสร็จแล้ว! นำไฟล์ไปตัดต่อวิดีโอ โพสต์ลง social media ส่งให้ทีม หรือเก็บไว้ใช้ทีหลังได้เลย
8.2 ↺ ล้างข้อมูลและเริ่มสินค้าใหม่ด้วยปุ่ม "↺ Reset" เมื่อต้องการเริ่มใหม่

ปุ่ม ↺ Reset อยู่ขวาของปุ่ม RUN — กดเพื่อล้าง output ทั้งหมดและเริ่มใหม่สำหรับสินค้าใหม่ ข้อมูลที่กรอกไว้ (ชื่อสินค้า, ราคา ฯลฯ) จะยังอยู่ แต่ผลลัพธ์ทั้งหมดจะถูกล้าง

⚠️ กด Reset หลังจาก Download ZIP แล้วเท่านั้น — ถ้ากดก่อน download ผลลัพธ์ทั้งหมดจะหายไป
